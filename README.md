# analytics project(pandas)

Анализ датасета с kaglle

Цель проекта: сегментировать покупатель с помощью rfm-анализа. Предложить стратегию по привлечению выделенных сегментов.

Шаги выполнения проекта:

1. Работа с данными: очистка данных, заполнение пропущенных значений, удаление дубликатов.
2. Базовая статистика для анализа датасета: минимум, максимум, медиана, распределение данных во времени, анализ выбросов.
3. Добавление категориальных значений и составление сводных таблиц по этим значениям.
4. Составление гипотезы и проверка гипотезы с помощью A/B тестов.
5. Проведение rfm-анализа и составление маркетинговых акции для привлечения отстающих пользователей.
